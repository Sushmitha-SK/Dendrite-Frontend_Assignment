# Frontend Assignment

This project was developed using

- **React Js with Typescript**
- **Bootstrap 5**
- **KeyCloak** for User Authentication
- **Redux** for State Management

## Problem Statement

Develop a simple web app to search music and create playlist, similar to Spotify
